JavaScript Fundamentals is made to help beginners get basic knowledge for JavaScript. JavaScript is an important part in web devleopent and any front-end developer should learn how to use JavaScript. Below are some topics you need to know about JavaScript, as they are used in almost every JavaScript problem.



FUNCTIONS

function random() {
                    for (word in array)
                    console.log(array[word])
                    }
                    random()

  • A function is a subprogram designed to perform a particular task.
  • Functions are executed when they are called. This is known as invoking a function.
  • Values can be passed into functions and used within the function.
  • Functions always return a value. If no return value is specified, the function will return undefined.
  • Functions are objects.



OBJECTS

let garden = {
                    vegetable: 'zucchini',
                    flower: 'sun flower',
                    fruit: 'grape',
                    water: true,
                    sun: true,
                    size: 10
                    }
                    console.log(garden.vegetable)

  • An object is a container that can hold multiple datatypes.
  • It has keys and values (color (key): 'blue' (value)), seperated with a colon.
  • Each key is seperated with a comma.
  • It will create and return an empty object, otherwise, it will return an object of a Type that
    corresponds to the given value.
  • If the value is an object already, it will return the value.
  • When called in a non-constructor context, Object behaves identically to new Object().



ARRAYS

 let days = ['Monday', 'Tuesday', 'Wednesday', 'Thursday', 'Friday']
                    console.log(days)
                    console.log(days.toString())
                    console.log(days.length)

  • In JavaScript, array is a single variable that is used to store different elements.
  • It is often used when we want to store list of elements and access them by a single variable.
  • Unlike most languages where array is a reference to the multiple variable, in JavaScript array is a
    single variable that stores multiple elements.
